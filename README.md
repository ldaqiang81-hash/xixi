# My Codex Pets

A personal collection of Codex companion pets.

## Pets

### Nuan (Nuan Nuan)
A warm, friendly chibi companion with warm peach skin, dark hair, and a cozy cyan scarf. v2 with 16-direction look tracking.

## Installation

Copy the pet folder to `~/.codex/pets/<pet-slug>/`.
